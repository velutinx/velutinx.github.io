#️⃣ Hashtag Validator

Check if a hashtag follows Twitter/X official format

✅ Twitter hashtag rules (syntax):

• Must start with # (if missing, we add it)

• Contains only letters, numbers, underscores, and Unicode letters (e.g. あ, カ, 中文, é)

• No spaces, punctuation, or special symbols (! @ $ % & * ( ) + = ?)

• Must include at least one letter (cannot be only numbers)

• Maximum length 140 characters (including #)

• Valid example: #FateGrandOrder , #クロエ・フォン・アインツベルン , #ChloeVonEinzbern

• Invalid: #123 (no letter), #hello!world (exclamation mark), #hashtag with space
